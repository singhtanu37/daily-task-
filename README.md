# Daily Task Tracker

A simple web app to add, complete, and delete tasks with persistent storage.

## Features
- Add tasks
- Mark tasks as complete
- Delete tasks
- Local storage (data saved even after refresh)

## Tech Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repo
2. Open index.html in browser
